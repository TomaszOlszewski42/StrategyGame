# StrategyGame
### Overview
This is a simple strategy game made for university course, which is also a reason why there is no commit history as it was part of larger repository.  
Game is played on hexagonal grid.
There are 3 countries, one controlled by player and 2 controlled by simple AI.
You can claim unowned provinces by moving there with an army and build markets to increase your income.
Other way to claim new land is to start war with other country and capture it's provinces.
You can save your progress to load it later.

### Engine
The game was made using Bevy Engine.

### Playing
Clone this repository and then use `cargo run` or `cargo run --release`.
